# 更新日志

本文件记录 CMTools 的版本更新历史。

## 关于例行版本发布

例行版本发布（如 2.8.2、2.8.3 等）通常意味着更新了内置的一个或多个分析工具。详细的工具变更请参阅各工具项目的发布说明。

---

## [2.8.7] - 2026-02-27

### 新增

- 并发控制优化：基于 CPU 线程数的动态并发限制
- 进程执行超时机制：120 秒超时保护，防止外部工具卡死
- 可执行文件智能缓存：仅在文件变更时重新写入

### 优化

- 前端本地化消息计算缓存，提升渲染性能
- 文件列表移除逻辑改进

### 重构

- Rust 代码格式化优化
- 前端代码清理

### 依赖

- 升级 Vite 7.x 及相关依赖
- 扩展 Tokio 依赖特性

### 文档

- 完善 CLAUDE.md 开发文档
- 添加 GitHub 社区健康文件（CONTRIBUTING.md、CODE_OF_CONDUCT.md 等）
- 更新 README.md 和 user_manual.md

## 版本演进

| 版本系列 | 时间跨度 | 主要特性 |
| ------- | ------- | ------- |
| v1.x | 2025.06-2025.07 | 基础功能搭建 |
| v2.0.x | 2025.07 | 迁移到 Tauri 2.0 |
| v2.1.x | 2025.07 | 界面重构、多语言支持 |
| v2.2.x-v2.3.x | 2025.08 | SMNFiler、UPDFiler 工具 |
| v2.4.x-v2.5.x | 2025.09 | 工具版本细分、多架构支持 |
| v2.6.x-v2.7.x | 2025.10-2026.01 | 工具迭代、跨平台优化 |
| v2.8.x | 2026.01-至今 | STR-Matcher 工具 |

**当前支持**：

- 8 个数据处理工具（AneuFiler、Aneu23、SMNFiler_v1/v2、UPDFiler_v1/v2、SHCarrier、STR-Matcher）
- 3 个平台（Windows、macOS、Linux）
