# 贡献者公约 行为准则 (Contributor Covenant 2.1)

## 我们的承诺

为了营造开放和友好的环境，我们贡献者和维护者承诺让参与本项目和社区的行为不受骚扰，无论年龄、体型、可见或不可见的残疾、种族、身体特征、性别认同或表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性认同和取向如何。

## 我们的标准

有助于营造积极环境的行为示例包括：

- 对他人展现出同理心和善意
- 尊重不同的意见、观点和经验
- 优雅地接受建设性批评
- 对造成的失误承担责任并道歉
- 关注对整个社区最有利的事情

不可接受的行为示例包括：

- 使用性化的语言或图像，以及不受欢迎的性关注或挑逗
- 嘲讽、侮辱性或贬低性评论，以及个人或政治攻击
- 公开或私下骚扰
- 未经许可发布他人信息，如物理或电子邮件地址
- 其他在专业环境中被合理认为不恰当的行为

## 执行责任

社区领袖负责 clarifying 和执行我们的行为标准，并将对任何他们认为不适当、具有威胁性、冒犯性或有害的行为采取适当和公平的纠正措施。社区领袖有权且有责任移除、编辑或拒绝不符合本行为准则的评论、提交、代码、维基编辑、问题和其他贡献，并在适当时指明需要修改的原因。

## 适用范围

本行为准则适用于所有社区场所，也适用于个人在公共场所代表社区时的情况。代表社区的方式包括使用官方电子邮件地址、通过官方社交媒体账户发布内容，或在线上或线下活动中担任指定代表。

## 执行

可以通过 GitHub Issues 或项目维护者联系方式报告滥用、骚扰或其他不可接受的行为。所有投诉都将得到及时和公平的审查和调查。所有社区领袖都有义务尊重任何事件报告者的机密性和安全性。

## 执行指南

社区领袖将遵循以下社区影响指南来确定对任何被认为违反本行为准则的行为的后果：

### 1. 纠正

**社区影响**：使用不恰当的语言或其他被社区认为不专业或不受欢迎的行为。

**后果**：社区领袖发出书面警告，说明违规行为的性质，并解释为什么行为不恰当。可能会要求公开道歉。

### 2. 警告

**社区影响**：通过单次事件或一系列行为构成的违规。

**后果**：警告并说明继续此类行为的后果。在警告期间不与涉及人员互动，包括主动与执行本行为准则的人员互动。这包括避免在社区场所和社交媒体等外部渠道的互动。违反可能导致临时或永久禁令。

### 3. 临时禁令

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定期限内禁止与社区进行任何形式的互动或公开交流。在此期间，不得与执行本行为准则的人员进行公开或私下互动。违反可能导致永久禁令。

### 4. 永久禁令

**社区影响**：表现出违反社区标准的行为模式，包括持续的不当行为、骚扰个人，或对某类人群进行攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。
